Step-by-Step Process

  1. Create a Database
    • Start your MongoDB shell or GUI tool.
    • Use use yourDatabaseName to create or switch to your database.
    • Confirm the active database.
  2. Insert Documents
    • Create a collection, e.g., db.collectionName.insertMany([{...}, {...}]).
    • Insert multiple documents with fields like name, age, address, etc.
  3. Find Documents
    • Display all documents: db.collectionName.find()
    • Find specific documents using filters: db.collectionName.find({name: 'John'})
  4. Update Documents
    • Locate a document with a filter, e.g., db.collectionName.updateOne({name: 'John'}, {$set: {age:30}})
    • Verify the update with a find command.
  5. Search Documents
    • Use queries with conditions, e.g., db.collectionName.find({age: {$gt:25}})
    • Review the matched documents.
  6. Delete Documents
    • Remove documents with db.collectionName.deleteOne({name: 'John'})
    • Verify deletion with a find command.
